# @glaeron/blender-mobile-3d

Secure installer and command-line manager for Blender Mobile 3D.

## Install

```bash
npx @glaeron/blender-mobile-3d install
```

## Commands

```bash
blender-mobile-3d install
blender-mobile-3d update
blender-mobile-3d uninstall
blender-mobile-3d doctor --json
blender-mobile-3d list-blenders
blender-mobile-3d version
blender-mobile-3d help
```
